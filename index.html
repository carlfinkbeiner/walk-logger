App Overview

Walk Logger is a SMS-based dog activity tracking application that allows users to log various aspects of their dog's daily routine through simple text messages. The app automatically timestamps activities and calculates walk durations, storing all data in a Google Sheets spreadsheet for easy tracking and analysis.

Key Features

üì± Supported Commands

Users can send any of these simple text commands:

Walk Start
Walk End
Poo
Pee
Feed

üîÑ How It Works

  1. User sends SMS: User texts any of the supported commands to the Twilio phone number
  2. Automatic timestamping: App captures the original SMS timestamp using Twilio's API
  3. Smart duration calculation: When "Walk End" is sent, app automatically calculates walk duration from the last "Walk Start"
  4. Data storage: All activities are logged to Google Sheets with date, time, action, and duration
  5. Confirmation: User receives a confirmation message with the logged time and duration (if applicable)

Data Structure

The app logs data in a Google Sheets spreadsheet with the following columns:

  • Date: MM/DD/YYYY format
  • Time: HH:MM format (24-hour)
  • Action: One of the five supported commands
  • Duration: Walk duration in minutes (only for "Walk End" entries)

Security & Privacy

  • All credentials stored as environment variables
  • Google Sheets API with service account authentication
  • Twilio webhook signature validation (can be enabled)
  • No personal data stored beyond activity logs
